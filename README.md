# 🔐 SafeVault AI

SafeVault AI is a web-based digital security application designed to protect users' sensitive data through secure authentication, intelligent risk detection, and emergency access control.

## 🚀 Key Features
- 🔐 Secure User Authentication (Firebase)
- ☁️ Real-time Cloud Data Storage (Realtime Database)
- ⚠️ AI-Powered Risk Analysis (Google Gemini Integration)
- 🚨 Emergency Lock System for instant protection
- 📂 Smart file management (metadata-based)

## 🧠 How It Works
Users can securely log in and manage their digital assets. The system monitors interactions and provides real-time security insights using AI. In case of suspicious activity, users receive alerts and can instantly lock the system to prevent unauthorized access.

## ⚙️ Tech Stack
- HTML5, CSS3, JavaScript
- Firebase Authentication & Realtime Database
- Google Gemini API (AI-based analysis)
